1 # Calduino
2 EMS Bus - Arduino library.
3 
4 **Calduino** provides functions to communicate through the **EMS Bus** with Buderus / Nefit / Worcester (or any other EMS Bus compatible) boilers. It includes commands for both getting status information (UBA Monitor, DHW Monitor, etc.) and setting new configurations (Set Day/Night Temperature, Set Working Mode, etc.).
5 
6 To know more about how the **EMS Bus** works have a look at this [post](https://domoticproject.com/ems-bus-buderus-nefit-boiler/). A full working Arduino sketch integrating Calduino and [WiFly](https://github.com/harlequin-tech/WiFlyHQ) libraries to connect wirelessly with the EMS Bus can be found in this [tutorial](https://domoticproject.com/calduino-connecting-arduino-ems-bus/).
7 
8 Doxygen documentation is available [here](https://danimaciasperea.github.io/Calduino/index.html).
9 
10 ## Requirements
11 To use this library you need two hardware components:
12 - An **EMS Bus – UART interface circuit** to convert the EMS Bus signals to UART TTL levels. Have a look at this [section](https://domoticproject.com/calduino-connecting-arduino-ems-bus#EMS_Bus_8211_UART_Interface_Circuit) to build your own circuit.
13 - An **Arduino Board** with Atmel ATmega microcontroller such as Arduino One or Mega. Development and debugging will be easier with more than one serial port, so I recommend **Arduino Mega 2560**.
14 
15 And of course you will need an EMS compatible boiler, as well as access to the EMS Bus.

21 ## Instalation
22 Install as any other [Arduino library](https://www.arduino.cc/en/Guide/Libraries): unzip the distribution zip file to the libraries sub-folder of your sketchbook.

24 ## Examples
25 Configure Calduino in the Serial Port 3 of Arduino Mega (pin TX 14 and RX 15) do:
26 
27  #include <Calduino.h>
28  #define DEBUG_UART_RATE 9600
29 
30  EMSSerial3.begin(EMS_BUS_UART_RATE);
31  calduino.begin(&EMSSerial3);
32 
33  Redirect the exit to Serial Port 0 (pin TX 1 / RX 0):
34 
35  #include <Calduino.h>
36  #define DEBUG_UART_RATE 9600
37  #define DEBUG_UART_RATE 9600
38 
39  EMSSerial3.begin(EMS_BUS_UART_RATE);
40  EMSSerial0.begin(DEBUG_UART_RATE);
41 
42  if (calduino.begin(&EMSSerial3, &EMSSerial0))
43  {
44  EMSSerial0.println(F("Calduino correctly started."));
45  }
46  else
47  {
48  EMSSerial0.println(F("Calduino error starting."));
49  }
50 Print EMS Datagram RC Datetime:
51 
52  calduino.printEMSDatagram(EMSDatagramID::RC_Datetime);
53 
54 Print configured DHW temperature:
55 
56  calduino.printEMSDatagram(EMSDatagramID::UBA_Parameter_DHW, DatagramDataIndex::selTempDHWIdx);
57 
58 Get current impulse temperature:
59 
60  float curImpTemp = calduino.getCalduinoFloatValue(FloatRequest::curImpTemp_f);
61 Set working mode in heating circuit 2 to night:
62 
63  calduino.setWorkModeHC(2, 0);
64 
65 Set night temperature in heating circuit 1 to 21.5℃:
66 
67  calduino.setTemperatureHC(1, 0, 43);
68 
69 Set DHW temperature to 50℃:
70 
71  calduino.setTemperatureDHW(50);
