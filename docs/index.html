EMS Bus - Arduino library.

This library provides functions to communicate through the EMS Bus with Buderus/Nefit/Worcester (or any other EMS Bus compatible) boilers. It includes commands for both getting status information (UBA Monitor, DHW Monitor, etc.) and setting new configurations (Set Day/Night Temperature, Set Working Mode, etc.).
